# 💳 UPI Trust & Payment Product Analytics

An end-to-end **Product Analytics** project that explores user trust, payment behavior, transaction success, and customer journeys within a UPI payment ecosystem.

This project demonstrates how product analytics can be used to understand user behavior, improve payment success rates, identify friction points, and support data-driven product decisions.

---

## 📌 About

The objective of this project is to analyze user interactions and payment transactions in a UPI platform to uncover insights that improve product performance and customer experience.

The project combines SQL, Python, Excel, and Power BI to perform data cleaning, exploratory analysis, KPI tracking, dashboard development, and business recommendations.
